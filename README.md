# Personal Portfolio

A clean, modern portfolio website built with pure HTML and CSS, featuring a dark editorial theme and deployed on Vercel.

## Features

- **Dark Theme**: Deep background with off-white text and electric blue accents
- **Typography**: Syne display font and JetBrains Mono for a distinctive look
- **Responsive Design**: Mobile-first approach using CSS Grid and Flexbox
- **Smooth Animations**: CSS-only fade-ins and hover effects
- **Semantic HTML5**: Proper structure for accessibility and SEO
- **Vercel Ready**: Configured for static site deployment

## Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. No build process required - it's pure HTML/CSS

## Deployment

1. Push this code to a GitHub repository
2. Connect your repo to Vercel at [vercel.com](https://vercel.com)
3. Deploy with zero configuration - Vercel auto-detects it as a static site

## Customization

- Update personal information in `index.html`
- Modify colors and fonts in `css/style.css` using CSS custom properties
- Add images to the `assets/` folder and reference them in the HTML

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- Google Fonts (Syne, JetBrains Mono)
